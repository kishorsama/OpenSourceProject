# The No-JS Recipe Book

## Overview
A simple static HTML recipe website with CSS-only features like dark mode toggle. No JavaScript is used.

## Project Structure
- `index.html` - Main homepage with recipe cards
- `Recipie/` - Individual recipe pages
  - `margherita-pizza.html`
  - `pancakes.html`
  - `quinoa-bowl.html`

## Development
The site is served using Python's built-in HTTP server on port 5000.

## Deployment
Configured as a static site deployment with the root directory as the public folder.
