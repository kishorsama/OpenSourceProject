# 🍲 Recipe Book Website

A clean, modern, and user-friendly web application for browsing and managing your favorite culinary recipes. This project is designed to be fast, responsive, and easy to navigate.

## ✨ Features
* **Dynamic Recipe Cards:** Beautifully displayed recipes with images and descriptions.
* **Search Functionality:** Real-time filtering to find recipes instantly.
* **Fully Responsive:** Optimized for mobile, tablet, and desktop screens.
* **Interactive UI:** Hover effects and smooth transitions for a premium feel.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure.
* **CSS3:** Custom styling and responsive Grid/Flexbox layouts.
* **JavaScript (ES6):** Dynamic content rendering and search logic.
